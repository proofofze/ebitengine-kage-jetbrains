## [0.1.0] - 2026-05-21

### Added
- Syntax highlighting for `.kage` files (keywords, types, builtins, numbers,
  comments) via a hand-written lexer.
- Code completion for keywords, types, and builtin functions.
- Quick documentation on hover for builtins and types.
- Brace matching and line/block comment toggling.
- Live templates (snippets): `fragment`, `vec2/3/4`, `mat2/3/4`, `imageSrcNAt`,
  `imageSrcTextureSize`, `imageSrcRegionOnTexture`, `clamp`, `mix`,
  `smoothstep`, `discard`.
- Configurable color settings page.
